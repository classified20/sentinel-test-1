# Sentinel App

Sentinel is a phone-tracking and personal safety application focused on reliable monitoring, proactive risk detection, and device-protection controls.

## Core Features

- **Live location tracking (online/offline):**
  - Real-time location updates when internet is available.
  - Offline location buffering and sync when connectivity is restored.
- **AI integration:**
  - Intelligent anomaly detection based on movement and context.
  - Smart alerts for unusual behavior patterns.
- **Risk evaluation:**
  - Continuous risk scoring from location, movement, and environment signals.
  - Risk-level notifications for trusted contacts or guardians.
- **Sentinel PIN before phone shutdown:**
  - The app prompts for a Sentinel PIN before allowing phone shutdown (where platform policies permit).
- **Bluetooth monitoring:**
  - Tracks trusted/unknown nearby Bluetooth devices as safety context.
- **Movement monitoring:**
  - Detects movement status changes (stationary, walking, driving, sudden stop).
  - Supports inactivity or irregular-motion alerts.

## Product Goal

Provide dependable location visibility, stronger safety intelligence, and tamper-resistance features to protect users in both connected and low-connectivity conditions.
